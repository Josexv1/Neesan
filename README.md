# Neesan

Electron + React + Webpack + Material UI

### Prerequisites

What things you need to install the software and how to install them


* Yarn
* Node 9+

### Installing

```
git clone https://github.com/Josexv1/neesan
cd neesan
yarn
yarn start
```
